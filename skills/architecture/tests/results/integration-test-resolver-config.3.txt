Set `Spec.ResolverRef.ResourceKind` to **`pipelinerun`** (lowercase `r`).

By default, this field is set to `pipeline`. You only need to change it to `pipelinerun` when you want to customize things that are defined at the PipelineRun level rather than the Pipeline level — such as:

- Pipeline timeouts
- Service accounts
- Workspaces

This is documented in `/tmp/konflux-docs/modules/testing/pages/integration/creating.adoc` (line 188), and the API reference confirms:

> `resourceKind` defines the kind of resource being resolved. It can either be `"pipeline"` or `"pipelinerun"` but defaults to `"pipeline"` if no value is set.
