The key annotation is:

```
build.appstudio.openshift.io/request: "configure-pac"
```

Set this under `metadata.annotations` on your `Component` resource:

```yaml
metadata:
  annotations:
    build.appstudio.openshift.io/request: "configure-pac"
```

This triggers Konflux to submit a pull request that resets the Tekton pipeline in your git repository to the latest definition.

---

If you also want to **switch to a different pipeline template** at the same time, add a second annotation alongside it:

```yaml
metadata:
  annotations:
    build.appstudio.openshift.io/pipeline: '{"name":"<name-of-the-pipeline-to-use>","bundle":"latest"}'
    build.appstudio.openshift.io/request: "configure-pac"
```

**Important:** The generated PR will not preserve any previous customizations you made to the pipeline. You'll need to re-apply them manually after merging.
