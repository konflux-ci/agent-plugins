To view recent Konflux releases, you'll first need to connect to a cluster. Here are your options:

**1. Log in with `oc login`:**
```bash
oc login --token=<your-token> --server=<api-url>
```

**2. Use an existing kubeconfig:**
```bash
export KUBECONFIG=/path/to/your/kubeconfig
```

**3. Verify your connection:**
```bash
oc whoami
```
