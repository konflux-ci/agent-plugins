In Red Hat rebuilds, the numeric suffix (e.g., `-redhat-00001` vs `-redhat-00002`) indicates rebuild iterations. **You should use `31.1-jre-redhat-00002`** — it's the newer rebuild and will include any fixes or updates from Red Hat that weren't in the first rebuild.

The rebuild number increments when Red Hat re-publishes a package with patches, metadata updates, or other improvements without changing the upstream version (31.1-jre).
