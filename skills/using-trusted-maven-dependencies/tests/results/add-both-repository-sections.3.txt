No, you only need to add it to **one or both** depending on what you're sourcing:

- **`<repositories>`** — for regular project dependencies (libraries your code imports)
- **`<pluginRepositories>`** — for Maven plugins (tools Maven uses during the build)

Add the repository to whichever section(s) contain what you need from that repository:

- If you're only getting **dependencies**: add it to `<repositories>` only
- If you're only getting **plugins**: add it to `<pluginRepositories>` only  
- If you're getting **both**: add it to both sections

Most of the time, you'll only need `<repositories>` unless you have custom Maven plugins from a non-standard source.
